# Pokedex

A Pokedex application built with NextJS + TypeScript + TailwindCSS + PokeAPI

## Setup

### Install Dependencies

```bash
npm install
```

### Run the Project

```bash
npm run dev
```

## How it works

It consumes pokeapi and populate the app.
